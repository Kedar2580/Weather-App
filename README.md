# Weather App 🌦️

A simple and responsive Weather App built using **HTML**, **CSS**, and **JavaScript** that fetches real-time weather data using the **OpenWeather API**.

---

## 🚀 Features

* Search weather by city name
* Displays:

  * Temperature
  * Weather condition
  * Humidity
  * Wind speed
* Dynamic weather icons
* Responsive design for mobile and desktop
* Real-time data from OpenWeather API

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the app
* **CSS3** – Styling and responsive UI
* **JavaScript (ES6)** – Functionality and API handling
* **OpenWeather API** – Weather data provider

## ⚙️ Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/Kedar2580/Weather-App.git
```

2. Open the project folder

```bash
cd weather-app
```

3. Get your API key from OpenWeather

[OpenWeather API](https://openweathermap.org/api?utm_source=chatgpt.com)

4. Add your API key inside `script.js`

```javascript
const apiKey = "YOUR_API_KEY";
```

5. Open `index.html` in your browser
